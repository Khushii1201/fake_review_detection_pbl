AI-Based Fake Review Detection

Using NLP, Logistic Regression & Fuzzy Logic

Khushi Wadhwa

Registration ID: 2427030511

Problem Statement

Fake reviews manipulate product ratings and mislead customers on e-commerce platforms. Manual detection is difficult and unreliable. There is a need for an automated intelligent system to detect fake and suspicious reviews.

Objectives

  • Detect fake reviews using Natural Language Processing (NLP)
  • Apply Machine Learning for classification
  • Integrate Fuzzy Logic for uncertainty-aware decisions
  • Deploy a web-based demonstration interface

Dataset

  • Amazon Reviews Dataset (Kaggle)
  • Reviews text and ratings
  • Converted ratings into binary labels
  • Balanced dataset for fair classification

NLP Preprocessing

  • Lowercasing
  • Punctuation removal
  • Stopword removal
  • Text cleaning and tokenization

Feature Extraction

TF-IDF (Term Frequency – Inverse Document Frequency)

  • Converts text into numerical vectors
  • Measures importance of words
  • Enables machine learning classification

Machine Learning Model

  • Algorithm: Logistic Regression
  • Binary Classification (Fake / Genuine)
  • Provides probability-based output
  • Balanced dataset accuracy ≈ 70%

Fuzzy Logic Integration

Instead of strict binary output:

  • 0 – 0.45 → Likely Genuine
  • 0.45 – 0.55 → Suspicious
  • 0.55 – 1 → Likely Fake

This handles uncertainty and improves interpretability.

System Architecture

User Input → NLP → TF-IDF → Logistic Regression → Probability → Fuzzy Logic → Final Output

Results

  • Successfully detects Genuine reviews
  • Identifies Suspicious reviews
  • Flags Likely Fake reviews
  • Provides confidence percentage

Implementation

  • Programming Language: Python
  • Libraries: Scikit-learn, NLTK, Gradio
  • Frontend: Web-based interface
  • Deployment: GitHub Pages

Conclusion

The system successfully integrates NLP, Machine Learning, and Fuzzy Logic to detect fake reviews with uncertainty-aware classification. It provides a scalable foundation for e-commerce review analysis.
